How to Use

  1. Upload the plant-sale-signs-data.*.zip file you received, then click Import.
  2. Review each plant in Step 3 — edit descriptions, adjust tags and categories, and click Mark Reviewed when done.
  3. Download the updated zip in Step 4 and pass it to the next reviewer (or return it to the coordinator).

Do not unzip or modify the zip file's contents — pass it as-is. Only the technical coordinator should work with the raw files inside.

Step 2 — AI Enrichment & Description Merge (Optional)

▶
AI Enrichment — fills descriptions for plants not in the uploaded CSV

Uses OpenAI gpt-4o with live web search. Prefers NCSU Plant Toolbox, Prairie Moon Nursery, USDA Plants, FSUS, and Missouri Botanical Garden. Cost: ~$0.03 per plant.

No API key? Use Copy prompt on the plant in Step 3, paste into ChatGPT or Claude, copy the JSON output, then click Fill from clipboard.

AI Description Merge — reconciles CSV descriptions against Squarespace data

Uses gpt-4o-mini (~$0.001/plant). Compares the CSV-derived description against Squarespace tags and description, removes contradictions, and produces a clean reconciled HTML description.

⚠ Security warning: Never enter an API key on a website you don't personally own and trust.
