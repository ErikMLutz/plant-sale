Step 2 — AI Enrichment & Description Merge (Optional)

AI Enrichment — fills descriptions for plants not in plants.csv

Uses OpenAI gpt-4o with live web search. Prefers NCSU Plant Toolbox, Prairie Moon Nursery, USDA Plants, FSUS, and Missouri Botanical Garden. Falls back to botanical training knowledge. Cost: ~$0.03 per plant (~$6 for a full 200-plant sale).

No API key? Use Copy prompt per row, paste into ChatGPT or Claude, copy the JSON, then click Fill from clipboard. Or edit fields directly in Step 3 — rows become "Manually Enriched."

AI Description Merge — reconciles plants.csv descriptions against Squarespace data

Uses gpt-4o-mini (no web search — text processing only, ~$0.001/plant). Compares the CSV-derived description against Squarespace tags and description, removes contradictions, and produces a clean reconciled HTML description. Can also be done per-row in Step 3.

⚠ Security warning: Never enter an API key on a website you don't personally own and/or trust — the site owner can steal it and charge API calls to your account. Only use this if you truly trust the site and know its source/origin.

Step 3 — Review & Edit

Plants marked Needs enrichment are missing data. Three ways to fill them in:

  1. Bulk auto-enrich / merge — enter an OpenAI API key in Step 2 and click the Enrich or Merge buttons.
  2. Per-row auto-enrich / merge — once an API key is entered, Auto-enrich and Auto-merge buttons appear on every row.
  3. Copy prompt + AI — click Copy prompt on any row, paste into ChatGPT or Claude, copy the JSON output, then click Fill from clipboard to apply it.
  4. Manual edit — type or edit HTML directly in the Description field. The row will be marked Manually Enriched.
