Step 2 — Review & Edit

Plants marked Needs enrichment are missing data. Three ways to fill them in:

  1. Auto-enrich — enter an OpenAI API key in Step 1.5 above and click Enrich. Runs automatically for all pending plants.
  2. Copy prompt + AI — click Copy prompt on any row, paste into ChatGPT or Claude, copy the JSON output, then click Fill from clipboard to apply the response.
  3. Manual edit — type directly into the Latin name, Attributes, Highlight, and icon fields. The row will be marked Manually Enriched.
