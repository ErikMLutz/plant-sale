Step 1.5 — AI Enrichment

Fills in Latin name, attributes, and highlight text for plants not found in plants.csv. Uses OpenAI gpt-4o with live web search. The prompt instructs it to prefer approved sources (NCSU Plant Toolbox, Prairie Moon Nursery, USDA Plants, FSUS, Missouri Botanical Garden) and fall back to botanical training knowledge when those sources don't have the plant.

Cost: ~$0.03 per plant (~$6 for a full 200-plant sale). For future years when you're only adding a handful of new plants, expect a few cents total. Use the Enrich 1 or Enrich 10 buttons to test before running the full batch.

Saving your work: After enriching, click Download enriched plants.csv in Step 2. Drop that file back into Step 1 next time and all previously enriched plants will load instantly — no API calls needed for plants already in the CSV.

The API key is optional — skip this section and fill in data manually in the review table, or use Copy prompt on any row to run the prompt yourself in ChatGPT or Claude and paste the result back with Fill from clipboard.

⚠ Security warning: Never enter an API key on a website you don't personally own and trust — the site owner can steal it and charge API calls to your account. Only use this if you downloaded this file and are opening it directly in your own browser (not via a shared URL).

Step 2 — Review & Edit

Plants marked Needs enrichment are missing data. Three ways to fill them in:

  1. Bulk auto-enrich — enter an OpenAI API key in Step 1.5 and click one of the Enrich buttons to run all (or a batch) of pending plants automatically.
  2. Per-row auto-enrich — once an API key is entered, an Auto-enrich button appears on every row. Use this to re-enrich a single plant or fix a bad result without re-running the whole batch.
  3. Copy prompt + AI — click Copy prompt on any row, paste into ChatGPT or Claude, copy the JSON output, then click Fill from clipboard to apply it. Works without an API key.
  4. Manual edit — type directly into the Latin name, Attributes, Highlight, and icon fields. The row will be marked Manually Enriched.
